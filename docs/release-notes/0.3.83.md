极序 Flow v0.3.83 更新说明

## 安全加固

1. Docker 端口绑定加固：MySQL 和 Backend 端口改为 127.0.0.1 绑定，避免公网直连；新增 `docker-compose.prod.yml` 生产覆盖配置。
2. 依赖漏洞修复：升级 multer 2.1.0 → 2.1.1、axios 1.13.6 → 1.16.0；通过 npm overrides 强制 path-to-regexp 0.1.13。
3. 新增 uuid 为显式依赖（原为通过 exceljs 传递解析）。
4. CI 安全门禁：新增 gitleaks 敏感信息扫描 + npm audit 依赖漏洞扫描，critical 级别阻断 PR。
5. Dependabot 集成：backend/frontend/desktop 三端每周依赖更新。

## 代码清理

6. 深度清理 Phase 2：删除 43 个文件（未引用组件、无用 hook、废弃脚本、旧构建产物），释放约 470MB。
7. 架构残留清理 Phase 3：恢复误删的 Dockerfile，修复全部悬空引用，验证三端构建通过。
