# v0.4.13

## 新功能
- **采购单可编辑**：草稿状态的采购单可直接修改供应商、仓库、明细等再提交，不用作废重建。
- **短装可结案**：供应商少发货时，采购单不再卡住——在采购单上「关闭剩余」即可按实收结案。

## 优化（采购结算更准确）
- **审核成为结算关口**：收货订单「审核通过」时才生成应付、完成采购单（以前是一上架就结算、审核形同虚设）。审核退回则暂不结算，处理后可重新审核。
- **应付按实收金额**：应付账单按实际入库数量 × 采购单价计算，短装时金额如实反映，不再固定按下单总额。

## 修复
- 修复重复生成应付账单的隐患（应付记录加唯一约束）。
- 修复快速点击「提交保存」可能重复创建采购单的问题（创建加幂等保护）。

## 说明
- 桌面端与网页端同步升级；桌面端仅随系统同步版本号，功能一致。
