# v0.4.7

## 稳定性修复
- 修复采购退货 / 调拨 / 盘点执行时的若干报错（SQL 占位符与字段约束问题）
- 退货 / 手动出库新增可用库存校验，防止扣减被销售单预留的库存导致超卖
- 数据库迁移兼容性增强（幂等化），全新部署与数据恢复更可靠

## 安全
- 清理前端与后端依赖中的高危漏洞

## 运维能力（服务端）
- 新增数据库每日自动备份（保留 14 天）
- 新增服务健康监控 + 钉钉告警，以及每日巡检报告（心跳）
- 新增自动化测试门禁（CI），每次发布前自动回归，并修复了一批"全新部署才会触发"的隐患

## 说明
本次桌面端功能与上一版一致，版本号随系统整体同步；核心改进集中在后端稳定性、安全与运维能力。
