## 界面简化

1. 销售管理页面状态列大幅精简：合并双行「销售/履约状态」为紧凑单行工作流状态，点击可直接跳转仓库任务。
2. 销售详情页术语优化：「履约进度」改为「作业进度」，去除生僻词汇。

4. 文档清理：删除过期的 Gitee 同步文档，修复部署指南中的过期内容，删除死链接。
