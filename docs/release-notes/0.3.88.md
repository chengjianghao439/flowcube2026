# FlowCube v0.3.88

## UI 视觉优化

- 全局圆角从 16px 缩减为 8px，去 AI 感
- 卡片和表格去掉阴影，更扁平干净
- 筛选区去掉外层卡片包裹，控件直接展示
- 表头去掉大写字母样式，中文更自然
- 页面背景从纯白改为极淡灰 (#f8f9fa)
- 状态 Badge 从胶囊形改为小圆角矩形

## 操作列重构

- TableActionsMenu 改为 SplitButton（主按钮 + 下拉箭头合并）
- 主按钮文字统一精简为两个字（占用、发货、查看、详情等）
- 操作列宽度统一缩减为 120px

## 图标与字体

- 全部 Emoji 替换为 Lucide 线条图标
- 等宽字体（font-mono）统一改为系统等线字体
- tabular-nums 全部去除

## 表格列

- 金额、状态、时间、操作列全部改为左对齐
- 列宽拖拽优化：缩放某列时最后一列反向补偿，表格始终撑满屏幕

## 开发体验

- 开发模式跳过 API 配置门控，Vite 代理自动处理
- 登录页开发模式下不显示 API 配置提示
