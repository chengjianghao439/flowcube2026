# FlowCube v0.3.92

## 销售单详情页重构

- 改为选项卡布局：订单信息 / 作业进度 / 取货明细 / 装箱进度 / 操作记录
- 基础信息合并物流信息，同行显示
- 商品明细增加型号、颜色列
- 作业进度显示取货数量
- 取货明细按条码展示扫描记录
- 装箱进度显示箱数汇总和每箱明细

## 出库看板移除

- 删除独立的出库看板页面，功能全部迁移至销售单详情
- 状态 Badge 点击改为跳转销售单详情

## 日期格式统一

- 全系统日期时间统一为 YYYY-MM-DD HH:MM 格式

## 子页面关闭优化

- 关闭子页面后自动跳回父级列表页

## 数据库变更

- 迁移 082：sale_order_items 新增 spec、color 列
