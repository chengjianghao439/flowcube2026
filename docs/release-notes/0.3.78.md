# FlowCube v0.3.78

- 隔离 PDA 与桌面系统入口，移除双方页面、菜单、按钮和工作台中的跨端跳转。
- 增加前端运行时跨端路由防护，防止普通客户端导航从 PDA layout 跳到桌面 layout，或从桌面 layout 跳到 PDA layout。
- 清理桌面工作区历史标签中的 PDA 路径，避免旧缓存恢复出跨端标签。
- 修复 PDA 自动更新固定下载地址可能命中旧 APK 缓存的问题。
- PDA 更新接口和下载接口改为 no-store，并为下载地址加入版本、versionCode 和文件时间戳。
- PDA 原生下载插件显式禁用 HTTP 缓存。
- PDA APK 版本提升到 1.1.22 / 24。
- PDA APK GitHub Actions 发布链补齐服务器上传、后端重建和线上接口校验。
