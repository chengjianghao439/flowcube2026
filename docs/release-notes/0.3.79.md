# FlowCube v0.3.79

极序 Flow v0.3.79 更新说明

本次更新重点提升 PDA 拣货、仓库任务、波次处理、打印任务和 PDA 设备识别的稳定性。

主要更新：

1. 优化 PDA 拣货扫码校验，后端会核对真实库存容器，减少扫错容器、扫错商品、扫错仓库带来的异常。
2. 优化仓库任务和波次状态保护，降低重复点击、并发操作导致的状态异常。
3. 优化打印任务完成、失败、重试回执处理，提升打印状态可靠性。
4. 增加 PDA 设备会话识别能力，为后续 PDA 设备管理做准备；当前阶段不会阻断旧 PDA 作业。
5. 废弃旧版手工拣货数量和手工复核接口，统一使用 PDA 扫码流程。
6. 补齐 PDA 设备会话数据库迁移，测试环境已可创建和识别 PDA 设备会话。

使用提示：

- 浏览器端请刷新页面后使用。
- PDA 端如页面异常，请退出后重新进入；如有 APK 更新提示，请按提示更新。
- 桌面端会自动检测更新，如未自动更新可重新打开应用。
- 本次更新通常不需要重新登录；如果遇到登录状态异常，请重新登录。
- 管理员可继续观察 PDA device session 日志，为后续启用强设备校验做准备。
