# v0.4.12

## 新功能
- 采购订单页新增「查询」弹窗，把搜索和筛选集中到一处，可按 **单号、备注、经办人、状态、供应商、仓库、产品、创建日期范围** 组合查询；供应商/仓库/产品支持弹窗搜索选择。
- 打开采购页默认只显示 **最近一周** 的订单，可在查询弹窗自行调整或清空查看全部。
- 列表顶部新增 **筛选摘要标签**，一眼看清当前筛选了什么，可逐项移除或一键清空。
- 采购列表在「创建时间」右侧新增 **订单备注** 列。

## 优化
- 状态筛选补上「已完成」，此前无法筛选已完成的采购单。
- 关键词搜索改为 **按采购单号** 精确匹配（供应商请用弹窗中的供应商选择器）。

## 修复
- 修复采购单导出 Excel 时 **未跟随搜索关键词** 的问题，导出结果现与列表所见一致。
- 修复取消采购单确认弹窗在个别情况下样式/文案判断不稳定的问题。
