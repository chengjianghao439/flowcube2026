## 页面精简

1. 移除「异常工作台」页面（原系统巡检 + 自动修复功能），该功能为数据一致性巡检兜底，系统稳定后不再需要。
2. 合并「审批与提醒」为「提醒中心」，仅保留财务与系统级通知提醒。
3. 三个工作台页面各司其职：岗位工作台（业务待办）、系统巡检（已移除）、提醒中心（通知提醒）。
4. 从所有页面中移除指向已删除页面的导航入口。

## 使用提示

- 浏览器端请刷新页面后使用。
- 桌面端会自动检测更新，如未自动更新可重新打开应用。
- 本次更新不需要重新登录。
