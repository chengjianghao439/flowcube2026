# FlowCube v0.3.90

## 商品管理重构

- 新增字段：SKU编码（自动生成）、货号（6位随机，可手动编辑）、颜色、供应商关联
- 型号字段放宽限制，从5字符扩展到100字符
- 供应商、单位、型号、颜色改为必填项
- 产品条码改为自动生成，前端不再显示
- 编码生成统一：删除旧版 generateCode.js，统一使用 codeGenerator.js

## 商品编辑独立页面

- 新增/编辑商品改为独立页面（/products/new、/products/:id）
- 使用 ActionBar + Section 卡片布局，与销售单页面风格一致
- 供应商、分类选择改为弹窗搜索模式（FinderTrigger + FinderModal）
- 销售价格改为可编辑输入，同时显示利润率
- 价格预览卡片附带加价百分比显示

## 价格管理

- 价格管理页面已移除，百分比显示集成到商品编辑页面

## 界面优化

- 表格状态 Badge 改为 outline 样式（与销售单一致）
- 表格列宽：移除进价和价格列，新增货号、型号、颜色、供应商列

## 使用提示

- 浏览器端请刷新页面后使用
- 本次更新需要运行数据库迁移 `npm run migrate`

## 数据库变更

- 迁移 081：product_items 表新增 sku_code、article_number、color、supplier_id 列
