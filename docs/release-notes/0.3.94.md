# FlowCube v0.3.94

## 标签模板系统

- 取消 ZPL/TSPL 双指令集并行，统一使用 ZPL
- 删除无用的旧版库存标签模板
- 新增「塑料盒标签」模板，B 前缀容器专用，不印数量
- 「库存条码标签」仅用于 I 前缀容器（整件库存）
- 「物流标签」增强：新增快递、运费方式、件数、装箱内容字段
- 「货架条码标签」「产品条码标签」保持不变

## 打印模板

- 所有订单接入打印模板：销售单、采购单、收货订单、退货单
- 采购单打印改为使用模板系统，不再用硬编码 HTML
- 收货订单详情页新增「打印」按钮
- 退货单列表新增「打印」操作

## ERP 端禁止收货

- 收货操作仅允许通过 PDA 完成
- 收货订单详情页移除收货入口

## 塑料盒管理

- 新增「塑料盒管理」页面（库存 → 塑料盒管理）
- 塑料盒绑定产品，一对一服务，清空后保留可再次使用
- 容器拆分支持转入已有塑料盒
- 塑料盒拆分默认不打印标签

## 其他

- 更新 CLAUDE.md 项目文档
