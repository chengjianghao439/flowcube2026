# FlowCube v0.3.36

## 摘要

**打印机兼容配置**：将 TSPL 兼容参数从“整机环境变量”升级为“按打印机保存”。现在每台标签机都可以单独配置 **字节编码**、**换行方式**、**CODEPAGE 行策略**，适配佳博、TSC 以及后续接入的其它 TSPL 机型，而不会相互影响。

**打印链路贯通**：打印机兼容设置会从 ERP 后端随打印任务下发，并传到桌面端本机 RAW 打印桥接；货架标签、PDA 箱贴和打印机管理页的 **TSPL 自检** 都会使用这组配置。

## 主要文件

- `backend/src/database/054_printers_tspl_compat.sql`：新增每台打印机的 TSPL 兼容字段。
- `backend/src/modules/printers/printers.service.js`：打印机读写兼容参数。
- `backend/src/modules/print-jobs/print-jobs.service.js`：任务响应附带打印机兼容配置。
- `desktop/lib/localPrint.js`：按打印机配置执行 TSPL 编码、换行与 CODEPAGE 策略。
- `frontend/src/pages/settings/printers/index.tsx`：打印机管理新增“兼容”配置入口。

## 升级说明

- **后端**：部署后需执行迁移，新增打印机兼容字段。
- **桌面端**：请安装本版，桌面端才会按打印机兼容配置实际出纸。
- 若生产环境仍使用同域 `/downloads/*.exe` 自动更新，请在服务器放置对应的 `0.3.36` 安装包后再让客户端升级。

## 相对 v0.3.35

从“为某台电脑临时调环境变量”升级为“任意打印机均可独立配置兼容策略”的正式方案。
