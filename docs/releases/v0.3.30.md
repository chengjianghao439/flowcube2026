# FlowCube v0.3.30

## 摘要

**标签 TSPL 画布**：默认模板中 **elements 画布** 与 ZPL 同源毫米坐标时，佳博等 **TSPL** 打印机可生成对应 TSPL（`SIZE` 与条码/文本次序一致）。

**打印预览缩放**：销售单全屏预览、**采购单** `PrintOrderDialog`、**商品条码** `BarcodeDialog` 均支持 **预览放大/缩小**（打印内容仍为原始版式）；打印模板编辑器增加 **画布缩放** 便于小编辑热敏标签。

## 前端

- `PrintPreviewZoomControls`：共用预览缩放控件。
- `PrintOrderDialog` / `BarcodeDialog`：预览区 `transform` 缩放；`printRef` 内 HTML 不变。
- `SaleOrderPrintTemplate`：销售预览改用共用缩放控件。
- `TemplateRenderer`：`displayScale`（销售预览 + beforeprint 复位）。
- `print-templates/editor.tsx`：画布 **35%～300%** 缩放，拖拽与网格随缩放换算。

## 后端

- `labelTsplTemplate.js`：`generateTsplFromElements`、`getLabelTsplFromDefaultTemplate` 读取 `paper_size`；标签高度与 **canvasHeightMm** 对齐（至 500mm）。

## 相对 v0.3.29

TSPL 补齐画布生成；各类单据/标签预览可读性增强。

## 升级说明

Web 部署更新前端即可；桌面端若需更新 `localPrint` 等已随上一版发布，本版以前端体验为主。
