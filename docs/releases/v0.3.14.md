# FlowCube v0.3.14

## 摘要

修复 **SSE 打印机编码大小写不一致** 导致「已连接 print-client 仍提示无在线客户端」、改进 **桌面端自动更新弹窗稳定性**，并保留与 **v0.3.13** 一致的打印/发版说明引用。

## 后端

- **`GET /api/print-jobs/listen/:printerCode`**：`listen` 使用中间件写入的 **`validatedPrinterCode`（数据库规范 code）** 注册 SSE，与 `pushToClients` / `getDispatchHintForJob` 一致。避免 MySQL 大小写不敏感校对下 URL 与库中 `code` 不一致时订阅记错键。
- **`collectSseSubscribersForPrinter`**：精确键无订阅时，按 **不区分大小写** 回退匹配 `sseClients`（兼容已建立、尚未重连的旧连接）。

## 桌面端（Electron）

- **`main.js`**：安装包首次更新检查延迟默认 **1800ms**（可用 **`FLOWCUBE_UPDATE_CHECK_DELAY_MS`** 覆盖），减少早于 `DesktopMessageBoxBridge` 挂载而丢失 IPC 弹窗的情况。

## 前端

- **`router`**：将 **`DesktopMessageBoxBridge` / `DesktopQuitDialog` / `ErpApiBaseHotkeyDialog`** 移出 **`ErpDesktopConnectionGate`**。在「正在连接服务器…」全屏等待时仍挂载桥接，主进程更新提示等 IPC 可正常送达。

## 升级说明

1. **后端**：部署后建议 **重启**；print-client **重连一次** SSE（重启脚本即可）。
2. **桌面**：重新 **`dist:win` 打包** 分发；或等待 CI 构建后从 Release 下载。
3. **权限**：打印工作站账号仍需 **`print:client`**（见 `042_print_client_permission.sql`）。

## 相对 v0.3.13

约 4 个业务文件 + 版本号/lockfile；不含仅 CI 文档的重复说明。
