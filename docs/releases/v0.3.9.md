# FlowCube ERP v0.3.9 更新说明

对应 Git 标签：`v0.3.9`（与 `desktop/package.json`、`frontend/package.json` 版本一致）。

## 更新内容（本版）

### 货架管理

- **删除**：列表支持删除货架；删除前校验：仍有库位「货架」字段指向该编码，或相关库位上仍有在库容器时禁止删除并提示原因。
- **新增扫码校验**：`POST /api/racks/scan-hint`，可扫 RCK / PRD / CNT 或商品编码，提示条码冲突或与货架在库绑定关系。
- **打印**：货架标统一走打印机管理中 **库存标签（inventory_label）** 绑定；修复缺库列/调度异常；接口返回精简字段（`jobId`、`printerCode` 等）。
- **界面**：操作列与「编辑 / 打印标」布局调整（表格 `DataTable` 操作列 sticky、行高、货架页按钮纵向堆叠）。

### 仪表盘与版本信息

- **更新说明**：GitHub Release 正文自动过滤 **Full Changelog** / compare 链接等冗余行；与同版本 `backend/downloads/latest.json` 中的 `notes` 合并择优展示。
- **`latest.json`**：补充面向用户的中文更新摘要，供内网或回退场景。

## 部署与配置

- 若尚未执行：**`backend/src/database/051_warehouse_racks_barcode.sql`**（货架 `barcode` 字段）。
- 标签打印：在 **打印机管理** 中为仓库绑定用途 **`inventory_label`（库存标签）** 的标签机。

## 自动更新与桌面端

1. 安装低于 **0.3.9** 的旧版后，确保 `GET /downloads/latest.json`（或你们配置的更新接口）返回本版本 `version` 与 `filename`。
2. 安装包名与 CI 一致：`FlowCube-ERP-Setup-0.3.9.exe`。
3. 推送标签 **`v0.3.9`** 触发 GitHub Actions **Build Desktop Installer**。
