# FlowCube v0.3.29

## 摘要

**TSPL 标签（佳博 / TSC 等）**：打印机表增加 `label_raw_format`（`zpl` | `tspl`），货架 / 容器 / 箱贴按绑定打印机生成 **TSPL** 或 **ZPL**；桌面端本机 RAW 同时接受两种格式；设置页可切换 **RAW** 指令集。

## 数据库

- **`053_printers_label_raw_format.sql`**：为 `printers` 增加 `label_raw_format`（默认 `zpl`）。部署后由 `migrate.js` 自动执行，或启动 API 时一并迁移。

## 后端

- **`labelTsplTemplate.js`**：TSPL 兜底布局与默认模板 `format: tspl` + `body` 占位符。
- **`print-jobs.service.js`**：按打印机 `label_raw_format` 生成 `contentType` 与内容。
- **`print-dispatch.js`**：`tspl` 与 `zpl` 均视为标签机类型。
- **`printers.service.js`**：创建/更新/列表暴露 `labelRawFormat`。

## 前端

- **打印机管理**：表格列 **RAW**，可选 ZPL / TSPL。
- **`desktopLocalPrint.ts`**：本机核销支持 `tspl` 与校验。

## 桌面端

- **`localPrint.js`**：RAW 校验支持 ZPL（`^XA`…`^XZ`）或 TSPL（含 `SIZE`、`CLS`、`PRINT`）。

## 相对 v0.3.28

佳博等 TSPL 打印机可绑定为 TSPL；斑马等仍为 ZPL。

## 升级说明

1. 拉代码后启动后端以执行迁移（或手动执行 `053`）。
2. 在「设置 → 打印机管理」将对应标签机 **RAW** 设为 **TSPL**。
3. 桌面端建议重新打包安装以更新 `localPrint.js`；仅 Web 则部署前端即可。
