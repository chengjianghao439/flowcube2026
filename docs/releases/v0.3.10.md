# FlowCube ERP v0.3.10 更新说明

对应 Git 标签：`v0.3.10`（与 `desktop/package.json`、`frontend/package.json` 版本一致）。

## 更新内容（本版）

### 登录与安全

- **登录态**：JWT 仅存 **当前浏览器会话**（`sessionStorage`），关闭标签页或应用后需重新登录。
- **本机记忆**：登录成功后默认记住**用户名**；勾选 **「在本机记住密码」** 时才会保存密码（`localStorage`，与 token 分离）。
- **ERP 登录页**与 **PDA 登录页** 文案与行为已对齐。

### 后端与货架

- **打印任务接口**：队列记录中的 `id`、`printerId`、`templateId`、`createdBy` 等字段统一为可 JSON 序列化的数字，避免 MySQL `BIGINT` 导致前端异常。
- **货架打印入队**：返回的 `jobId` 与上述一致。
- **货架列表**：操作列改为单行横向滚动，按钮文案「打印标」简化为「打印」，避免小屏换行挤压。

## 部署与配置

无新增数据库迁移（相对 v0.3.9）。

## 自动更新与桌面端

1. 安装低于 **0.3.10** 的旧版后，确保 `GET /downloads/latest.json`（或你们配置的更新接口）返回本版本 `version` 与 `filename`。
2. 安装包名与 CI 一致：`FlowCube-ERP-Setup-0.3.10.exe`。
3. 推送标签 **`v0.3.10`** 触发 GitHub Actions **Build Desktop Installer**。
