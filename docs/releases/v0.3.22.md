# FlowCube v0.3.22

## 摘要

**桌面端**：修复 **Windows 下中文打印机名** 导致 ZPL 本机 RAW 打印 **`OpenPrinter` 失败** 的问题——打印机名称改由进程环境变量 **`FC_PRINTER_NAME`**（UTF-16）传入 PowerShell，避免经命令行编码损坏；**`print-zpl-raw.ps1`** 兼容 **PowerShell 5.1**，并简化 C# 侧异常以提升 **`Add-Type`** 编译兼容性。**自动更新**由系统原生弹窗改为 **全局应用内对话框**（任意页面含登录页均可提示）；**仪表盘 → 系统版本** 在检测到新版本时提供 **「立即更新」** 按钮，与主进程下载安装流程一致。

## 桌面端（Electron）

- **`lib/localPrint.js`**：Windows RAW 打印通过 **`FC_PRINTER_NAME`** 传打印机全名；**`execFileSync`** 使用 buffer 解码便于排错。
- **`lib/print-zpl-raw.ps1`**：参数精简为 **`ZplPath`**；从环境变量读取打印机名；**`Win32Exception`** 单参数形式。
- **`lib/updateCheck.js`**：支持 **`ui: 'ipc'`**，向渲染进程发送 **`flowcube:update-available`**（延迟约 2s）；导出 **`ignoreVersion`**。
- **`main.js` / `preload.js`**：IPC **`get-app-version`**、**`is-packaged`**、**`start-update-download`**、**`ignore-update-version`**；自动检查使用 **IPC 模式**。

## 前端

- **`GlobalDesktopUpdateDialog`**：挂载于路由根，订阅主进程更新推送；**立即更新 / 忽略此版本 / 稍后提醒**。
- **`DashboardVersionCard`**：安装包下展示 **Electron 应用版本**；有新版本且可解析下载地址时显示 **「立即更新」**。
- **`appVersionCompare`**、**`resolveAppUpdateDownloadUrl`**：与主进程一致的版本比较与安装包 URL 解析。

## 升级说明

1. **桌面**：安装本版本 **Windows 安装包**（见 GitHub Release 附件或贵方 **`/downloads/`** 静态资源）。
2. **服务端**：若使用 **`GET /downloads/latest.json`** 驱动自动更新，请将 **`version`**、**`filename`**、**`notes`** 更新为与本发行说明一致（仓库内已维护 **`backend/downloads/latest.json`** 示例）。

## 相对 v0.3.21

Windows 中文打印机名 RAW 打印稳定性；桌面自动更新提示改为全局应用内 UI；仪表盘一键触发更新下载。
