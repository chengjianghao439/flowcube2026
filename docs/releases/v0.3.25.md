# FlowCube v0.3.25

## 摘要

**标签打印模板**：类型 5–9 改为与销售单相同的 **拖拽画布**（条码 + 文字），不再手写 ZPL；**后端**根据 `layout.elements` 与纸张宽度 **生成 ZPL**（203dpi），并 **兼容** 旧版 `format=zpl` 存储。

**界面**：打印模板 **编辑页** 使用 **PageHeader + 卡片** 布局，与设置类页面一致；列表 **编辑 / 删除** 按钮样式与仓库等页面统一。

## 后端

- **`labelZplTemplate.js`**：`generateZplFromElements`、`getLabelZplFromDefaultTemplate`（画布或 legacy ZPL）；无有效片段时回退内置 ZPL。
- **`print-jobs.service.js`**：使用 `getLabelZplFromDefaultTemplate` 替代仅替换 ZPL 正文。
- **`print-templates.service.js`**：标签类型校验 **elements 非空** 或兼容 **format=zpl**。
- **`migrate.js`**：默认标签种子改为 **`elements`** 画布结构。

## 前端

- **`print-templates/editor`**：标签类型使用与销售模板相同的画布；左侧字段含条码/文字；**恢复默认布局**；旧 ZPL 模板打开时提示并切换为默认画布。
- **`labelZplDefaults.ts`**：`DEFAULT_LABEL_ELEMENTS`、`LABEL_PREVIEW_SAMPLE`。
- **`print-template.ts`**：元素类型含 **`barcode`**。
- **`TemplateRenderer`**：条码元素占位渲染。

## 升级说明

1. **后端**：部署并重启；若需新默认种子且库中已有旧种子，可手动编辑模板或重新保存默认项。
2. **桌面**：安装 GitHub Release 中的新安装包；**`backend/downloads/latest.json`** 已指向本版本（在线更新元数据）。

## 相对 v0.3.24

标签模板可视化编排；打印页与列表操作样式对齐；ZPL 由画布生成并保留旧数据兼容。
