# FlowCube v0.3.31

## 摘要

**本机 RAW 标签（桌面端）**：向 Windows 提交 **TSPL** 前将换行统一为 **CRLF**，改善部分 TSC/佳博固件「队列成功但不出纸」的情况；主进程日志增加 RAW 下发字节数便于排查。

**Web 提示**：货架标签、装箱箱贴在「本机核销」成功时，文案改为明确 **已提交 RAW 并核销队列**，并提示核对 ZPL/TSPL 与系统打印队列，减少「显示成功却未见出纸」的误解。

## 变更文件

- `desktop/lib/localPrint.js`：TSPL CRLF、`[FlowCube RAW]` 日志。
- `frontend/src/pages/racks/index.tsx`、`frontend/src/pages/pda/pack.tsx`：成功提示文案。

## 升级说明

- **桌面端**：须安装本版（含 `localPrint` 逻辑）后 TSPL 换行修复生效。
- **Web**：部署前端即可更新提示文案。
- 若使用 **`GET /downloads/latest.json`** 驱动自动更新，请将 `version` / `filename` / `notes` 指向本版本。

## 相对 v0.3.30

本版以本机标签 RAW 可靠性及用户提示为主，无数据库迁移。
