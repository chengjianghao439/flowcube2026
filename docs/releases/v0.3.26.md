# FlowCube v0.3.26

## 摘要

**标签打印模板**：编辑页支持 **手动设置纸张宽×高（mm）**（含快捷 80×200、58×150），不再仅用热敏下拉；保存时写入 **`layout.canvasWidthMm` / `canvasHeightMm`**，画布与打印 **ZPL `^PW`** 宽度一致。

**后端**：`generateZplFromElements` 优先使用 `canvasWidthMm`（30–120 mm）；`validateLayout` 校验可选宽高范围。

## 前端

- **`print-templates/editor`**：标签类型显示宽、高数字输入与快捷按钮；单据类型仍为纸张规格下拉。

## 后端

- **`labelZplTemplate.js`**：`resolveLabelWidthMm` 与画布宽度对齐。
- **`print-templates.service.js`**：标签布局可选 `canvasWidthMm` / `canvasHeightMm` 校验。

## 升级说明

部署后端并刷新前端；已有标签模板无宽高字段时仍按 `paper_size` 推断，保存后即写入新字段。

## 相对 v0.3.25

标签纸张可自定义毫米尺寸并与打印宽度对齐。
