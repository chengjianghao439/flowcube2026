# FlowCube v0.3.28

## 摘要

**前端本机 ZPL 打印**：区分「非桌面端 / 无桥接」与「接口未带回可打印载荷」，避免在桌面端已就绪但缺少 ZPL 或 `jobId` 时仍提示「已加入打印队列」，造成「界面显示成功而 Windows 打印队列无任务」的误解。

## Web / 桌面壳内前端

- **`frontend/src/lib/desktopLocalPrint.ts`**：`DesktopLocalPrintResult` 拆分为 `skipped_no_desktop` / `skipped_no_payload`（原笼统 `skipped`）。
- **`frontend/src/pages/racks/index.tsx`**：对上述两类分别 `toast.warning` 并中止后续「入队成功」提示。
- **`frontend/src/pages/pda/pack.tsx`**：装箱打印同样处理，避免误报「箱贴已加入打印队列」。

## 升级说明

- **仅 Web 访问**：部署含本版本的前端静态资源即可。
- **桌面安装包**：若需与 `package.json` 版本号一致，可重新执行 `desktop` 构建并随 Release 分发；本版主要为前端逻辑，未改 Electron 主进程与 `print-zpl-raw.ps1`。

## 相对 v0.3.27

本机打印「跳过 RAW」时的提示与误报修复（货架标签、PDA 箱贴）。
