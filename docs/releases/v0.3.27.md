# FlowCube v0.3.27

## 摘要

**桌面端 Windows RAW 打印**：`print-zpl-raw.ps1` 在 `WritePrinter` 成功后增加 **写入字节数与 ZPL 长度一致** 校验，避免部分写入仍被当作成功提交。

## 桌面端

- **`desktop/lib/print-zpl-raw.ps1`**：`written != bytes.Length` 时抛出异常，便于发现传输不完整；需重新安装/更新桌面端后生效。

## 升级说明

以本仓库 **GitHub Release** 中的新 **Windows 安装包** 覆盖安装 FlowCube 桌面端；仅使用 Web 端、不依赖本机 ZPL RAW 时可不更新。

## 相对 v0.3.26

Windows 本机 ZPL RAW 写入完整性校验。
