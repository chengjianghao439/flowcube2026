# FlowCube ERP v0.3.11 更新说明

对应 Git 标签：`v0.3.11`（与 `desktop/package.json`、`frontend/package.json` 版本一致）。

## 更新内容（本版）

### 打印机管理（桌面端）

- **添加打印机**：仅在 **FlowCube 桌面客户端** 中，从 **本机操作系统已安装打印机列表** 中选择（与系统「打印机」设置一致），并选择类型后添加；提交前会再次枚举本机列表校验。
- **移除**：手写名称/编码的添加与编辑表单；表格中「编辑」入口已去掉。
- **Electron**：主进程通过 `getPrintersAsync` 枚举；`preload` 暴露 `getSystemPrinters()`。
- **后端**：`POST /api/printers` 支持写入 `source: local_desktop`；新增迁移 `052_printers_source.sql`（若库中已有 `source` 列可跳过 ALTER，仅需保证 `db_migrations` 已记录）。

### 其他

- 桌面端更新检查、主进程与预加载相关调整；Toast、布局与标签页、货架列表与库位表单等体验与接口小优化。

## 部署与配置

- **数据库**：若尚未有 `printers.source` 列，请执行 `backend/src/database/052_printers_source.sql`（或依赖后端启动时的迁移；列已存在时勿重复 `ALTER`）。
- **自动更新**：部署后端静态资源 `GET /downloads/latest.json` 指向本版 `version` 与 `filename`（见仓库 `backend/downloads/latest.json`）。
- **安装包**：CI 产物名与历史一致：`FlowCube-ERP-Setup-0.3.11.exe`。
- **发版**：推送标签 **`v0.3.11`** 触发 GitHub Actions **Build Desktop Installer**，并创建/更新 **GitHub Release**（上传 `.exe`）。
